Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an elements in the DOM such as getElementsByClassName, getElemntsById, getElementsByTagName, querySelector, querySelectorAll etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: represents the HTML content inside an eliment. innerText: represents the visible text inside an element. textContent: represents all text content, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event starts from the target element to parent element.

Question-5: How do you create, add, and remove elements using JavaScript?

To Create an element: we can use document.createElement("tagName"). To Add an element: we can use appendChild(). To Remove an element:we can use element.remove().
